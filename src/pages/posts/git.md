## git 簡介
- 定義: 分佈式版本控制軟體(Distributed Version Control System, DVCS)
    - 分布式：只可以將程式依照功能、模塊拆分成不同的分支(branch)，交給不同的工程師進行開發，當工程師開發完後，可以在分支提 PR (Pull Request) 請求將改動合併(merge) 到主分支(master)。
    - 版本控制：
- 用途：
    - 管理版本
    - 便於協作


## git 常用指令
- 下載倉庫到本地 
```bash
    git clone <>
```
- git checkout <branch>
- git diff
- git add
- git commit -m <提交改動>
- git push
- 提 PR
